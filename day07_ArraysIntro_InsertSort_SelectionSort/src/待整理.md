
# 数组去重


我感觉有缺陷，必须在排序后使用？

```java
//去重
for (int i = 0; i < arr.length; i++) {
    if (i == arr.length - 1){
        System.out.println(arr[i]);
}else if (arr[i] != arr[i + 1]){
    System.out.println(arr[i]);
    }
}
```

```java
//倒排去重，输出结果为倒
int[] arr2 = new int[count];
for (int i = arr.length - 1; i > 0; i--) {
    if (i == 1) {
        count--;
        arr2[count] = arr[i];
    } else if (arr[i] != arr[i -1]) {
        count--;
        arr2[count] = arr[i];
    }
}
```
